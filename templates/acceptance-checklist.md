# 视觉项目验收清单

## 输入和环境

- [ ] 系统、内核、架构和运行时版本已记录
- [ ] 摄像头设备、分辨率、像素格式和实际采集帧率已记录
- [ ] 光照、背景、目标距离/姿态和测试布置已记录
- [ ] 使用的 GPIO/UART/模型/标定数据来源已记录

## 算法和坐标

- [ ] 目标检测/测量/跟踪在固定输入上可重复
- [ ] 坐标系、单位、方向、原点和标定方法已写明
- [ ] 无效检测、丢失、遮挡和重捕获状态已定义
- [ ] 误差计算与题目评分指标一致

## 接口和安全

- [ ] 通信帧、频率、超时和失败动作来自题目或用户确认
- [ ] 执行机构有初始状态、限幅、低速/空载测试和急停
- [ ] 激光、云台、电机、无人机等风险已隔离
- [ ] 未把下位机职责和 RK3566 职责混淆

## 性能和验收

- [ ] 采集 FPS、处理 FPS、发布频率和端到端延迟已分开记录
- [ ] 静态误差或动态误差达到题目要求，数据来自目标板
- [ ] 完成必做项和所声明发挥项测试
- [ ] 已记录连续运行时间、异常次数和降级行为
- [ ] 待实测项目没有被写成已完成结论
